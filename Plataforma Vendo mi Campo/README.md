# Plataforma Vendo mi Campo
Plataforma orientada a la compra-venta de parcelas de agrado.

### Descripción
Plataforma que sirve como base de datos y gestión en la venta o arriendo de los terrenos en la línea de negocios Campos (aquellos que solo permiten el cultivo, no la construcción), esta base de datos es mostrada en el sitio web de la empresa vendomicampo.cl a través de varios webservices.
